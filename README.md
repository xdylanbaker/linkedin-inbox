# linkedin-inbox

Automate LinkedIn inbox management and smart messaging workflows.

## Examples

```bash
# Process unread messages
openclaw skill linkedin-inbox process --unread-only

# Auto-reply to connection requests
openclaw skill linkedin-inbox auto-reply --template follow-up --filter role:engineer
```

## Installation

```bash
openclaw skills install xdylanbaker/linkedin-inbox
```

## Local Development

```bash
cp -r skills/linkedin-inbox ~/.local/share/openclaw/skills/
```
